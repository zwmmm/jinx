## createGlobalModal

创建一个全局modal，不需要自己挂载，但是需要增加一个Context（建议使用）
```tsx
import { VVModalProvider } from 'vvmodal'
function App(props) {
  return (
    <VVModalProvider>
      {props.children}
    </VVModalProvider>
  )
}
```
